# Finance-Manager
A personal finance manager built with Flask and MySQL, featuring secure auth, income/expense tracking, savings goals, and analytics dashboards. Frontend uses HTML, CSS, and JavaScript. Includes CSV/PDF export, profile management, and clean REST APIs for seamless interaction.
for running the website u need to follow these steps
step1-extract all files
step2- install all required libraries and mysql.
step3- create database
step4-open app.py code and find line"app.config['SQLALCHEMY_DATABASE_URI'] = 'mysql+pymysql://root:Your_password@127.0.0.1:Your_port/Your_databse_name'"
step5-replace your_password with your password of mysql database and your_database_name with your database name.
step6-give permissions for database to edit.
step7-run the app.py code.
